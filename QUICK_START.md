# ⚡ QUICK START - 10 MINUTES TO RUNNING SERVER

## For Complete Beginners

### Step 1: Open Terminal (2 minutes)
1. Press `Windows + R`
2. Type: `cmd`
3. Press Enter

### Step 2: Navigate to Project (1 minute)
```bash
cd C:\Users\iraki\Documents\OurWebInfo
```

### Step 3: Install Dependencies (3 minutes)
```bash
npm install
```
**Wait for it to finish** - you'll see messages scrolling

### Step 4: Start Development Server (1 minute)
```bash
npm run dev
```

✅ **You should see:**
```
Server running on port 3000
```

### Step 5: Test It Works (2 minutes)
1. Open browser
2. Go to: `http://localhost:3000/api/users/profile`
3. You should see:
```json
{"message":"User profile data"}
```

### ✅ SUCCESS! Server is running!
